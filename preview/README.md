# Preview Images

This folder contains preview screenshots of the RPEmotes BetterUI interface.

## Required Images

To properly display the preview in the main README, please add the following images:

1. **main-menu.png** - Screenshot of the main menu showing all categories
2. **emote-selection.png** - Screenshot of the emote grid/selection view
3. **emote-detailed.png** - Screenshot showing detailed emote view with multiple emotes

## How to Add Images

1. Take screenshots in-game (F4 to open menu)
2. Save them as PNG files with the names above
3. Place them in this `preview/` folder
4. Commit and push to GitHub

The images will automatically display in the main README once uploaded.
