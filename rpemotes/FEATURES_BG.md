# 🌟 Характеристики на новия UI

## 🎨 Визуален дизайн

### Футуристичен стил
- **Неонови cyan/teal цветове** - Ярки, модерни и привлекателни
- **Светещи ефекти** - Всички елементи имат glow ефект
- **Прозрачен фон** - Blur ефект за по-добра видимост
- **Закръглени ъгли** - Модерен и гладък дизайн

### Анимации
- **Плавно отваряне** - Slide-in анимация при отваряне
- **Hover ефекти** - Всички бутони реагират на hover
- **Светещи линии** - Анимирани линии в header-а
- **Smooth transitions** - Всички промени са плавни

## 🔍 Функционалност

### Търсачка
```
┌─────────────────────────────────┐
│  🔍 Търси емоут...              │
└─────────────────────────────────┘
```
- Търсене в реално време
- Филтриране по име и описание
- Работи с всички категории
- Бързо и responsive

### Категории
```
┌──────┬──────┬──────┬──────┐
│ Всички│ Танци│ Емоути│ Споделени│
└──────┴──────┴──────┴──────┘
```
- Лесна навигация
- Визуално разделение
- Active state индикация
- Scroll за много категории

### Grid система
```
┌────────┬────────┬────────┬────────┐
│ 🎭     │ 💃     │ 🐶     │ 📦     │
│ Emote1 │ Dance1 │ Animal1│ Prop1  │
└────────┴────────┴────────┴────────┘
```
- Responsive grid layout
- Auto-fill колони
- Hover ефекти на картите
- Икони за всеки тип емоут

## ⌨️ Клавишни комбинации

| Клавиш | Действие |
|--------|----------|
| `F4` | Отваряне/Затваряне на менюто |
| `ESC` | Затваряне на менюто |
| `ENTER` | Избор на емоут (при hover) |
| `Typing` | Автоматично фокусиране на търсачката |

## 🎯 Типове емоути

### 🎭 Обикновени емоути
- Всички стандартни емоути
- Описание с команда `/e (name)`
- Филтриране по категории

### 💃 Танци
- Специална категория за танци
- Икона 💃 за лесно разпознаване
- Всички dance емоути на едно място

### 👥 Споделени емоути
- Емоути за двама играчи
- Автоматично търсене на близък играч
- Описание с `/nearby` команда

### 😊 Изражения (Moods)
- Facial expressions
- Persistent между респавни
- Лесно reset

### 🚶 Стилове на ходене
- Различни walking styles
- Persistent между респавни
- Команда `/walk (style)`

### 📦 Prop емоути
- Емоути с обекти
- Специална икона 📦
- Texture variations (ако има)

### 🐶 Животински емоути
- Емоути за животни
- Показват се само за non-human peds
- Специална икона 🐶
