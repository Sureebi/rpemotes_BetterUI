# Инструкции за инсталация на новия UI

## Стъпка 1: Проверка на файловете

Уверете се че имате следните нови файлове:

```
rpemotes-reborn/
├── html/
│   ├── index.html
│   ├── style.css
│   ├── script.js
│   └── color-themes.css
├── client/
│   └── EmoteMenuNUI.lua
└── fxmanifest.lua (обновен)
```

## Стъпка 2: Backup на стария UI

Преди да продължите, направете backup на:
- `client/EmoteMenu.lua` (стария файл)
- `NativeUI.lua`
- `fxmanifest.lua` (оригинала)

## Стъпка 3: Рестартиране на ресурса

1. Отворете F8 конзолата в играта
2. Напишете: `restart rpemotes-reborn` (или името на вашия ресурс)
3. Натиснете Enter

## Стъпка 4: Тестване

1. Натиснете **F4** (или вашия конфигуриран клавиш)
2. Трябва да видите новото футуристично меню с cyan цветове
3. Тествайте:
   - Търсачката
   - Категориите
   - Избор на емоут
   - Затваряне с ESC

## Възможни проблеми и решения

### Проблем: Менюто не се отваря

**Решение:**
1. Проверете F8 конзолата за грешки
2. Уверете се че `ui_page 'html/index.html'` е в `fxmanifest.lua`
3. Проверете че всички HTML файлове са в `html/` папката

### Проблем: Менюто е празно (няма емоути)

**Решение:**
1. Проверете че `EmoteData`, `SharedEmoteData`, `ExpressionData` и `WalkData` са заредени
2. Уверете се че `client/AnimationList.lua` се зарежда преди `client/EmoteMenuNUI.lua`

### Проблем: Стилът не се зарежда правилно

**Решение:**
1. Изчистете кеша на FiveM (изтрийте `cache` папката)
2. Рестартирайте играта
3. Проверете че `style.css` е в `html/` папката

### Проблем: Искам да се върна към стария UI

**Решение:**
1. Отворете `fxmanifest.lua`
2. Променете:
```lua
-- Премахнете:
ui_page 'html/index.html'
files {
    'html/index.html',
    'html/style.css',
    'html/script.js'
}

-- Променете в client_scripts:
'client/EmoteMenuNUI.lua' -> 'client/EmoteMenu.lua'

-- Добавете обратно:
'NativeUI.lua' (в началото на client_scripts)
```
3. Рестартирайте ресурса

## Персонализация

### Промяна на цветовете

1. Отворете `html/color-themes.css`
2. Изберете желаната тема
3. Копирайте кода
4. Поставете го в края на `html/style.css`
5. Рестартирайте ресурса

### Промяна на заглавието

В `html/index.html` намерете:
```html
<h1 class="menu-title">EMOTE MENU</h1>
```

Променете на:
```html
<h1 class="menu-title">ВАШЕТО ЗАГЛАВИЕ</h1>
```

### Промяна на текстовете

В `html/index.html` можете да промените:
- Placeholder на търсачката: `placeholder="Търси емоут..."`
- Footer текстове: `<span class="key">ESC</span> Затвори`

## Допълнителни настройки

### Промяна на размера на менюто

В `html/style.css` намерете:
```css
.menu-container {
    width: 900px;
    max-height: 85vh;
    ...
}
```

Променете `width` и `max-height` според вашите предпочитания.

### Промяна на броя колони в grid-а

В `html/style.css` намерете:
```css
.emotes-container {
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    ...
}
```

Променете `200px` на по-малка стойност за повече колони или по-голяма за по-малко колони.

## Поддръжка

1. Проверете конзолата (F8)
2. Проверете server конзолата
3. Уверете се че всички зависимости са инсталирани
